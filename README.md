# Mining log parse
 xjb写的脚本用来分配矿池的收益

## Usage

> 确保log内容格式为`[YYYY-MM-DD HH:mm:ss.sss]  miner    speed 10s/60s/15m SPEED1 SPEED2 SPEED3 H/s max SPEED4 H/s`

```bash
python3 parse.py log1.log log2.log ...
```
